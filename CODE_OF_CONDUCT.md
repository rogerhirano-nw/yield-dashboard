# Code of Conduct

This repository is an internal tool used by the Newsweek yield / ad-ops team.
We expect everyone working on it — whether full-time staff, contractors, or
external collaborators — to follow a basic standard of professional conduct.

## Expected behavior

- Be respectful, direct, and constructive in code review, issues, and chat.
- Assume good intent. Ask before assuming something is broken or wrong.
- Keep discussion focused on the work, not the person doing the work.
- Disagree on the merits. Escalate to the team lead if a disagreement stalls.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks of any kind.
- Sharing credentials, API keys, internal financial data, or PII outside the
  authorized team.
- Knowingly committing unreviewed changes that bypass the team's normal PR /
  approval workflow on production-facing branches (`main`, `mac-studio`).

## Reporting

If you see behavior that violates this standard, raise it with your manager
or the repository owner. Reports will be handled confidentially.

## Scope

Applies to all repository interactions: code, issues, pull requests, commit
messages, review comments, and any related team communications.
